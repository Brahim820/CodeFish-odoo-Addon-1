Odoo Website File Upload

Odoo Website files upload is an additional feature which provides your customers to upload attachments on shop

Key Features:

  • Customers can upload attachments/files on payment page.
  • Customers can upload any type of attachments/files (i.e: jpg, png, pdf, .doc, .xls).
  • Customers can upload as many as attachments/files with no limit.

How to use

Customer can upload attachments/files on payment page.


Customers can upload any type of attachments/files.
